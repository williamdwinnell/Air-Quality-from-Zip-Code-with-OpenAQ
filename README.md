# Air-Quality-from-Zip-Code-with-OpenAQ
Get the air quality of a Zip Code with OpenAQ in python.

# Dependencies
The following libraries should be installed before attempting to run the code.

`pip install geopy`

# Example Output
```
{
    "o3": {
        "value": 0.03,
        "unit": "ppm",
        "quality": "Very Good"
    },
    "pm25": {

        "quality": "Moderate"
```
